Chatbox

Chatbox is a simple utility to transform what you type in a chat box into bold, italic, or other fonts, allowing you to express emotion or emphasis.

By lala (Bondage Club member 47249)

Usage

If the input line starts with a space, Chatbox is disabled.

If you go back and insert a space at the start of the line, all the Chatbox changes are undone, back to just before Chatbox made the first change to the input line.

Installation

The recommended method of installation is the first one of the methods below which applies to you

Using TamperMonkey

  1. Install TamperMonkey (Chrome, Firefox, Microsoft Edge)
  2. Open this page
  3. Confirm script installation

With a bookmark (not Android)

  1. Copy this text:

    javascript:(function(){let s=document.createElement("script");s.setAttribute("language","JavaScript");s.setAttribute("crossorigin","anonymous");s.setAttribute("src","https://lalabcpurple.github.io/chatbox/chatbox.js?_="+Date.now());s.onload=()=>s.remove();document.head.appendChild(s);})();

  2. Make a bookmark with that link
  3. Go to Bondage Club
  4. Click the bookmark

With a bookmark (Android)

  1. Copy the above text
  2. Make a bookmark with that link
  3. Go to Bondage Club
  4. Type the name of the bookmark into the URL bar
  5. Select the entry with a ☆ on the left

In a javascript console

  1. Copy the above text
  2. Go to Bondage Club
  3. Paste it into a javascript console
