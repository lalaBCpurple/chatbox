Installation

With a bookmark

  1. Copy this link
  2. Make a bookmark with that link
  3. Go to Bondage Club
  4. Click the bookmark

In a javascript console

  1. Copy this link
  2. Go to Bondage Club
  3. Paste it into a javascript console
